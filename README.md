# FairPlate

App para dividir cuentas en grupo. Desarrollada con Ionic + Angular.

- Permite agregar personas e items
- Asignar personas a cada item
- Propina porcentual, se divide proporcionalmente al subtotal de cada uno
- Resumen con lo que debe cada persona

## Requisitos

- Node 18+
- Ionic CLI

## Correr en Android

ionic cap add android
ionic cap run android
